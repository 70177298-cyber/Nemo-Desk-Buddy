# Nemo-Desk-Buddy

### Components Required

- 128x64 Oled Display
- MPU6050 3axis gyro sensor
- Touch Sensor
- 3d printed Customize box
- ESP32C3 Super mini
- TP4056 Battery Charging module
- 3.7v Lithium Battery
- small sliding switch


### Wire Connection

- Oled Display Wiring connection
- SDA GPIO 4
- SCL GPIO 5
- VCC 3.3v
- Gnd to gnd

### MPU5060 Wiring connection
- SDA GPIO 4
- SCL GPIO 5
- VCC 3.3v
- Gnd to gnd

### Touch Sensor
- I/o GPIO 3
- VCC to 5v 
- Gnd to gnd

# Nemo-Desk-Buddy
# Components Required

# 1: 128x64 Oled Display 
# 2: MPU6050 3axis gyro sensor
# 3: Touch Sensor
# 4: 3d printed Customize box
# 5: ESP32C3 Super mini
# 6: TP4056 Battery Charging module
# 7: 3.7v Lithium Battery
# 8: small slideing switch

# Wire Connection

# Oled Display Wiring connection
# SDA GPIO 4
# SCL GPIO 5
# VCC 3.3v
# Gnd to gnd

# MPU5060 Wiring connection
# SDA GPIO 4
# SCL GPIO 5
# VCC 3.3v
# Gnd to gnd

# Touch Sensor
# I/o GPIO 3
# VCC to 5v 
# Gnd to gnd
